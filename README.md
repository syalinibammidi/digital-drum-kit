# 🥁 Digital Drum Kit

An interactive Digital Drum Kit built using **HTML, CSS, and JavaScript**.

## Features

* Keyboard-controlled drum pads
* Mouse click interaction
* Real-time sound generation using the Web Audio API
* Modern responsive UI
* Beat recording functionality
* Playback of recorded beats
* Visual button animations

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Web Audio API

## How to Run

1. Download or clone the repository.
2. Open `index.html` in a web browser.
3. Use the keyboard or click the drum pads to play sounds.
4. Record and replay your own beat sequence.
