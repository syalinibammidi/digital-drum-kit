# 🥁 Digital Drum Kit

An interactive Digital Drum Kit built using **HTML, CSS, and JavaScript**. This web application allows users to play drum sounds using keyboard keys or mouse clicks, with an additional beat recording and playback feature for an enhanced interactive experience.

## ✨ Features

* Keyboard-controlled drum pads
* Mouse click interaction
* Real-time sound generation using the Web Audio API
* Beat recording and playback
* Visual button animations
* Modern responsive user interface

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Web Audio API

## 🚀 How to Run

1. Clone or download this repository.
2. Open `index.html` in your preferred web browser.
3. Use the keyboard or click the drum pads to play sounds.
4. Record your own beat sequence and replay it.
